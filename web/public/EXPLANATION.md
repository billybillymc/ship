# web/public/

Static assets served directly by the web server without processing.

## Files

- **manifest.json** -- PWA web app manifest defining app name, icons, and display mode.
- **robots.txt** -- Search engine crawler directives.
- **browserconfig.xml** -- Windows tile configuration for pinned sites.
- **og-image.png** -- Open Graph image used for social media link previews.

## Directories

- **icons/** -- App icons in various sizes for PWA and browser integration.
