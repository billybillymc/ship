# web/src/components/ui/

Low-level reusable UI primitives.

## Files

- **Combobox.tsx** -- Searchable single-select dropdown built on Radix Popover and cmdk, with optional clear/none option.
- **ContextMenu.tsx** -- Positioned right-click context menu with keyboard navigation and submenu support.
- **ErrorBoundary.tsx** -- React error boundary catching render errors with a "Try Again" fallback UI.
- **Skeleton.tsx** -- Animated placeholder loading skeletons for various layouts (list, kanban, editor, team table).
- **TabBar.tsx** -- Horizontal tab bar with accessible markup, active underline indicator, and right-side content slot.
- **Toast.tsx** -- Toast notification system with auto-dismiss, success/error/info variants, and optional action buttons.
- **Tooltip.tsx** -- Styled tooltip wrapper around Radix UI's Tooltip primitive.
- **ScrollFade.tsx** -- Gradient fade overlay at the bottom of scrollable containers.
- **PropertyRow.tsx** -- Standard layout for labeled property fields in sidebars, with optional tooltip and missing-field highlighting.
- **MultiAssociationChips.tsx** -- Multi-select chip input for document associations with searchable dropdown and colored removable chips.
- **ScrollFade.test.tsx** -- Tests for the ScrollFade component.
