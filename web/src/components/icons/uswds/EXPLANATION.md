# web/src/components/icons/uswds/

U.S. Web Design System (USWDS) icon integration.

## Files

- **Icon.tsx** -- USWDS icon component that lazy-loads SVG icons via Vite's glob import, with runtime name validation and ARIA accessibility attributes.
- **types.ts** -- TypeScript type definitions for USWDS icon names.
- **index.ts** -- Barrel exports for the Icon component and types.

## Directories

- **__mocks__/** -- Test mocks for icon loading.
