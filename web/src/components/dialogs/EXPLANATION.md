# web/src/components/dialogs/

Modal dialog components for complex user interactions.

## Files

- **BacklogPickerModal.tsx** -- Bulk-select issues from the backlog to add to a sprint, project, or program via checkbox multi-select with search filtering.
- **ConversionDialog.tsx** -- Confirmation dialog for converting documents between issue and project types, explaining consequences.
- **MergeProgramDialog.tsx** -- Dialog for merging one program into another with entity preview, conflict warnings, and type-to-confirm safety step.
