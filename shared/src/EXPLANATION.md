# shared/src/

Shared TypeScript types and constants used by both API and web packages.

## Files

- **index.ts** -- Barrel export re-exporting all types and constants.
- **constants.ts** -- Shared HTTP status codes, error code strings, and session timeout values (15-minute idle, 12-hour absolute per NIST AAL2).

## Directories

- **types/** -- TypeScript interface definitions.
